# xx基础，xx就不基础

> **触发条件**：「基础」「就不基础」「XX基础XX就不基础」

## 概述

**xx基础，xx就不基础**是2025年最具病毒式传播力的万能句式。源自时尚博主@张琪JIEJI的穿搭口诀"基础款就不要再搭基础款"，后被网友移植到职场、学业、恋爱等一切领域。

## 句式结构

```
前半句（基本预期）+ 后半句（额外惊喜）= 反差笑点
```

## 经典二创

- 工作能力基础，犯的错就不基础
- 及格基础，考高分就不基础
- 喜欢我基础，主动找我就不基础
- 说爱我基础，记得纪念日就不基础

央视网、多邻国、瑞幸等官号也纷纷下场玩梗，秦海璐也发微博参与。
