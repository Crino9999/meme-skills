### 鸡煲文化（中文社区最活跃梗文化）

- "鸡煲" = 故障机器人，"机宝"谐音，隐含调侃其弱
- 「鸡煲还在启动」— 描述机器人启动慢，被精英怪锤爆
- 「偏差认知」— 一张掉集中的卡，衍生文字递减梗："那一天的集中 / 那一天的集中 / 那一天的 / 那一天 / 那 / -1"
- 「回响形态」— 复读梗
- 「第四强的角色」— 总共四个角色，意思是垫底
- B站主播「病院坂saki」以长期直播鸡煲A20挨打闻名，cos鸡煲唱歌

## 三、重大事件：「差评风波」（2026年3-4月）

### 时间线
1. **3月6日**：EA上线，初期好评如潮（~96%）
2. **3月20日**：首个Beta补丁削弱大量流派核心卡牌
3. **3月21日**：一天内差评从2300激增至16000+，简中区首爆
4. **3月21日**：Mega Crit发长文回应中国玩家
5. **4月3日**：Casey Yano受访表示"不因差评生气"
6. **截至4月底**：累计差评超1.5万，简中"多半差评"

### 差评核心原因
1. "削弱无限"触及中国玩家核心乐趣
2. EA阶段痛点未解，先砍爽点
3. 中国玩家缺乏Discord/Reddit等反馈渠道，Steam评测是"唯一能被听见的方式"
4. "环国区好评"——全球好中国差，形成落差

### 衍生文化
- **"环国区好评"** — 成为新梗，指全球好评唯独中国差评
- 贴吧中外玩家水平大战
- 日媒GameSpark专门报道
- Yano回应原文"那咋了 他们不是玩家吗？"被广泛引用

## 七、对SKILL.md编写的关键建议

1. **称呼**：正式名=「杀戮尖塔2」，对话中用「尖塔2」「塔2」「爬塔」更自然
2. **角色名**：使用官方中文译名 + 括号标注社区昵称，如"故障机器人（鸡煲）"
3. **差评风波**：需保持中立——这是沟通渠道受限导致的社区现象
4. **鸡煲文化**：必须覆盖「偏差认知」「回响形态」「还在启动」「第四强」等核心梗
5. **"爬塔"语义**：既是动词也是名词
6. **敏感点**：避免贬低中国玩家的反馈方式
7. **参考资源**：优先引用灰机Wiki和游民星空
