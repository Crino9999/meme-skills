## 三、角色阵容（EA 首发 5 个）

### 回归角色（3个）

| 角色 | 变化要点 |
|------|----------|
| **Ironclad（铁甲战士）** | 保留强攻/力量核心，更强调 Exhaust 协同 |
| **Silent（静默猎手）** | 弃牌机制大改，新增 **Sly** 关键词；小刀流加入 Exhaust |
| **Defect（故障机器人）** | 球体系大幅调整，新增第4种球体类型 |

> ⚠️ **观者（The Watcher）** EA 首发完全缺席。

### 新增角色（2个）

#### Necrobinder（灵魂绑定者）
- **设定**：刻薄巫妖，使用活化左手 **Osty（奥斯蒂）** 作战
- **核心机制**：
  1. **Osty** — 独立骷髅手同伴，有HP，可召唤/增强/牺牲
  2. **Doom（厄运）** — 层数≥敌人HP时即死执行（无视格挡）
  3. **Souls（灵魂）** — 0费代币牌，抽2后消耗
- **起始遗物**：Bound Phylactery
- **特色**：全游戏机制最复杂角色

#### Regent（摄政者）
- **核心机制**：**Stars（星辰）** 资源 + **Forge（锻造）** 君王之刃
- 可在跑局中制作特定卡牌（非随机奖励）

## 四、核心玩法变化（vs STS1）

| 方面 | STS1 | STS2 |
|------|------|------|
| 引擎 | LibGDX (Java) | **Godot** |
| 角色数 | 4 | **5 个 EA 首发** |
| 多人模式 | 无 | **4人联机合作** |
| 地图结构 | 固定路线 | **分支章节（Alternate Acts）** |
| 遗物升级 | 无 | **篝火可强化遗物（Forge Relic）** |
| 卡牌附魔 | 无 | **Enchantments（附魔）系统** |
| 新卡牌类型 | — | **Quest Cards（任务牌）** |
| 敌人AI | 固定意图 | 动态，Boss多阶段变身 |
| Ascension | 1-20级 | 简化为 **1-10级** |
| 模组支持 | Java限制 | Godot原生支持 |
| 剧情 | 碎片化 | Timeline/Epoch系统 |

### 2026年3月 "削卡风波"
- Mega Crit 移除强力卡牌互动循环（"无限loop"）
- 中文区玩家发起差评轰炸
- 官方回应并部分回调被删卡牌
- 形成"环国区好评"现象

- Steam 2026年3月最畅销游戏
