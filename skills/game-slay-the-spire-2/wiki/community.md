## 三、鸡煲文化（核心梗群）

"鸡煲"是中文社区最活跃、最持久的梗文化，从一代延续到二代：

### 「鸡煲」的由来
- "鸡煲" = 「机宝」谐音
- 隐含调侃故障机器人强度偏弱——"菜的像个鸡煲"
- 社区最著名的迫害对象，有专门的"鸡煲笑话"流派

### 五大核心梗

**1. 「还在启动」**
- 描述机器人启动速度慢，经常被精英怪锤爆
- 典型句式：「鸡煲还在启动，人已经没了」
- 暗示机器人的构筑需要较多回合才能成型

**2. 「偏差认知」**
- 指机器人一张会持续减少集中（Focus）的卡牌「偏差认知」
- 玩家将集中值递减做成文字递减梗：
  - "那一天的集中 / 那一天的集中 / 那一天的 / 那一天 / 那 / -1"

**3. 「回响形态」**
- 机器人的「回响形态」卡牌衍生出复读梗
- "于是人们开始复读——于是人们开始复读"

**4. 「第四强的角色」**
- 一代总共四个角色（铁甲 / 猎人 / 机器人 / 观者）
- "第四强" = 垫底的意思
- 典型句式：「鸡煲是一代第四强的角色」（实际就四个）

**5. 病院坂saki（B站主播）**
- 以长期直播鸡煲 A20（最高难度）挨打闻名
- 曾 cos 鸡煲唱歌
- 自称"全网最尊重故障机器人的主播"
- 代表了一代鸡煲文化的延续

## 四、差评风波（2026年3-4月）

这是二代上线后最重大的中文社区事件，直接影响了游戏的 Steam 评价格局。

### 时间线

| 日期 | 事件 |
|------|------|
| 2026-03-06 | EA 上线，初期好评如潮（好评率 ~96%） |
| 2026-03-20 | 首个 Beta 补丁上线，**削弱大量流派核心卡牌** |
| 2026-03-21 | **一天内差评从 2,300 激增至 16,000+**，简体中文区首先爆发 |
| 2026-03-21 | 简中好评率暴跌至 ~52-67%，全语言跌至 79% |
| 2026-03-21 | Mega Crit 发布**长文回应中国玩家**，承诺改善沟通渠道 |
| 2026-03-28 | 部分回调，但争议持续 |
| 2026-04-03 | Casey Yano 接受 PC Gamer 采访："不因差评生气"，理解中国玩家 |
| 2026-04-18 | 0.103.2 版本推送主分支，简中好评率 52%，全语言跌至 68% |
| 2026-04 底 | 累计差评超 1.5 万，简中仍为"多半差评" |

### 差评核心原因

1. **"削弱无限"触及根本**：中国玩家将"无限"构筑视为终极目标与游戏乐趣核心，官方更新直接削弱无限并表态"让无限更难达成"——被视作对核心玩家群体的背刺
2. **痛点未解，先砍爽点**：EA 阶段存在的精英怪过强、部分卡牌超模等问题未被修复，反而先砍玩家喜欢的流派
3. **沟通渠道有限**：中国玩家无法使用 Discord / Reddit 等西方主流社区，Steam 评测是"唯一能被听见的方式"（Casey Yano 原话承认）
4. **"环国区好评"的落差**：其他语言区好评率远高于中文区，形成鲜明对比和失落感

### 衍生现象

- **「环国区好评」**——成为中文游戏社区新梗，描述"全球好评唯独中国差评"的现象
- **贴吧中外玩家论战**——讨论"中外玩家水平差异"，部分中国玩家认为自身更追求强度/最优解
- **日媒报道**——日本 GameSpark 专门发文"中国玩家差评为何如此多"，引发跨文化讨论
- **「那咋了 他们不是玩家吗？」**——Casey Yano 回应原文，部分认为诚恳，部分认为避重就轻

### 评价现状（截至 2026 年 5 月）
