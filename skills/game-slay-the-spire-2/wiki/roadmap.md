# 未来路线图

> 信息来源：2026 年 4 月 17 日 Neowsletter 第 21 期。Mega Crit 明确表示**不设固定时间表**。

## 开发理念

创始人 Casey Yano：

> "Exacting deadlines produce sloppy uninspired work, and I don't want Sloppy Spire 2, I want Slay the Spire 2."
>
> "严格的时间表只会产出粗糙无灵感的作品，我不想要 Sloppy Spire 2，我只要 Slay the Spire 2。"

核心原则：**质量优先，无固定时间表。**

## 已确认内容

| 类别 | 计划 | 说明 |
|------|------|------|
| 新角色 | 1 个新角色 | 可能是观者（The Watcher）回归，也可能是全新角色 |
| 分支章节 | Act 2 / Act 3 的 Alternative Acts | 目前仅 Act 1 有分支章节（每章两个不同区域，敌人池 / Boss 不同） |
| 新内容 | 更多卡牌、事件、遗物、药水 | 持续扩充游戏内容池 |
| Steam Workshop | 官方模组工坊 | Godot 引擎原生模组支持，比一代（Java/LibGDX）大幅提升 |
| 实验游戏模式 | 非传统游玩模式 | 具体内容未公开 |
| Steam 成就 | 成就系统 | 含集换式卡牌 |
| "True Victory" | 真正的结局 | 完整叙事终局内容 |
| 平台移植 | 主机 + 移动端 | EA 结束后启动 |
| Badge / Score | 重制奖章 / 分数系统 | 含好友排行榜 |
| 本地化 | 更多语言支持 | 具体语种未公布 |
| 最佳图鉴 | 游戏中查看敌人信息 | 便捷的资料查阅系统 |

## EA 周期预期

| 问题 | 官方回答 |
|------|---------|
| EA 预计多长？ | **1-2 年**（参考：初代 EA 约 1.5 年） |
| 正式版价格？ | 正式版发布后价格**会上涨** |
| 当前完成度？ | "功能基本完整"，部分新卡 / 道具使用占位美术 |
| 社区反馈渠道？ | Discord、Steam 讨论、游戏内 **F2 键反馈工具**、匿名化数据收集 |
| 测试分支？ | Steam → 属性 → 测试版 → `public-beta`，更新更频繁 |
